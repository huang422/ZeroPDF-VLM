## 1. FieldSchema
**Source**: [vlm_pdf_recognizer/recognition/field_schema.py](../../vlm_pdf_recognizer/recognition/field_schema.py)

```python
@dataclass
class FieldSchema:
    field_id: str           # e.g. "VX1", "person1", "year"
    field_type: str         # one of: title | version | checkbox | stamp | text | number | person_number
    template_id: str        # contractor_1 | contractor_2 | enterprise_1
```

| `checkbox` | ‚ùå | ‚úÖ | AIP |

| `text` | ‚úÖ | ‚úÖ | AIP |
| `number` | ‚úÖ | ‚úÖ | AIP |
| `person_number` | ‚úÖ | ‚úÖ | AIP |

## 4. RecognitionResult
**Source**: [vlm_pdf_recognizer/recognition/vlm_recognizer.py](../../vlm_pdf_recognizer/recognition/vlm_recognizer.py)

Per-field output of `VLMRecognizer._recognize_field`.

```python
@dataclass
class RecognitionResult:
    field_id: str
    has_content: Optional[bool]      # True / False / None (title fields, AIP unavailable)
    content_text: Optional[str]      # VLM output post-cleanup, or None
    raw_response: str                 # Raw VLM bytes (debug)
    parse_success: bool               # False after exhausting retries
    inference_time_ms: float          # VLM wall-clock (0 if skipped)
    retry_count: int = 0              # 0..3
    AIP_has_content: Optional[bool] = None
    AIP_ink_ratio: Optional[float] = None
    AIP_component_count: Optional[int] = None     # legacy, currently always None
    AIP_time_ms: Optional[float] = None
    processed_roi_image: Optional[np.ndarray] = None  # diff image from AIP
    VLM_content_text_allPdf: Optional[str] = None     # reserved, unused in current pipeline
```

**Invariants** enforced by `validate()`:
- `has_content == False` ‚üπ `content_text is None`.

**JSON projection** (`to_json_dict()` of the parent):
- AIP_* fields are **not** emitted.
- `has_content` and (for non-checkbox fields) `content_text` are emitted.
- `processed_roi_image` is saved separately to `processed_rois/` PNGs.

```python
    def validate(self, template_schema: TemplateSchema) -> None: ...
    def calculate_results_status(self) -> bool: ...
    def to_json_dict(self) -> Dict[str, Any]: ...
```

### `calculate_results_status()` algorithm

```
1. If field_id="VX1" exists and has_content == True:
       return False                         # disagreement overrides everything
2. date_valid =
       True if no fields in {year, month, date} are present,
       else any(field.has_content for those fields)
3. Build other_fields = all field_results where:
       has_content is not None             (skip titles)
       and field_id not in {VX1, year, month, date}
       and field_type != "version"
4. other_fields_valid = all(f.has_content for f in other_fields)
5. return date_valid AND other_fields_valid
```

**Note**: `VX2` is included in `other_fields` and MUST be `True` to pass. This is the "agreement checkbox" ‚Äî the user must affirmatively check it.

## 6. Case-Level Aggregation
**Source**: [vlm_pdf_recognizer/output.py](../../vlm_pdf_recognizer/output.py)

```python
REQUIRED_TEMPLATE_TYPES = {"contractor_1", "contractor_2", "enterprise_1"}

def _aggregate_case_results(vlm_results) -> Dict[case_id, info]:
    # info shape:
    # {
    #   "case_results": bool,
    #   "document_count": int,
    #   "valid_count": int,
    #   "invalid_count": int,
    #   "template_types_present": sorted list,
    #   "missing_template_types": sorted list,
    #   "non_target_documents": sorted list,
    # }
```

A case is valid iff:
1. `all_valid` ‚Äî every doc has `results=True`.
2. `all_targets` ‚Äî every doc's `template_id` ‚àà `REQUIRED_TEMPLATE_TYPES`.
3. `all_types_present` ‚Äî every type in `REQUIRED_TEMPLATE_TYPES` appears at least once.

`"unknown"` or `"error"` template IDs (from failed Feature 001 matches) trigger `non_target_documents` and bust both `all_targets` and `all_types_present`.

## What's Intentionally Absent From This Feature

- **HuggingFace model loading / quantization** ‚Äî replaced by Ollama (`vlm_loader.py` only manages the Ollama client).
- **JSON-structured prompting** ‚Äî replaced by raw-text OCR + post-processing pipeline.
- **Per-document CSV** ‚Äî replaced by per-date CSV.
- **Manual `auxiliary_has_content` / SIFT-based comparison** ‚Äî superseded by AIP (Feature 004).
